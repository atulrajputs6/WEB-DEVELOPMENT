Semantic ElementsIn HTML, Semanitc elements are elements that provide meaning to the structure of a web page, makeing it easier for search engines and assitive technologies to understand the content. They are used to define the different parts of a web page, such as header, footer, article, main content, navigation and more.

Example of Semantic Elements

  1. Header
    Used to define the haeder section of a web page.
  2. Footer
    Used to define the footer section of a web page.
  3. Article
    Used to define an article or a piece of content.
  4. Nav
    Used to define the navigation section of a web page.
  5. Aside
    used to define a piece of content that is related to te main content, but not essential.
  6. Figure
    Used to define a figure or image
  7. Details
    Used to define a collapseable section of content that can be toggled open or close by the user.
  8. Summary
    Used to define or prove brief summary of main content.
  9. Figcaption
    Used to define caption for a figure or an image.
  10. Dialog
    Used to create a dialog box to display important information or warning
